HabitFlow es una aplicación web minimalista y elegante diseñada para el seguimiento de hábitos diarios.
Centrada en la experiencia de usuario y el diseño visual, permite a los usuarios construir rutinas sólidas con una interfaz oscura moderna y un sistema de progreso intuitivo.

Características Principales
- Onboarding Personalizado: Sistema de bienvenida que guarda el perfil del usuario localmente.
- Gestión de Hábitos: Añade, marca como completados o elimina hábitos con un límite preventivo de 10 tareas para fomentar el enfoque.
- Visualización de Progreso: * Barra de progreso dinámica con degradados de color según el éxito.
- Anillo de progreso circular interactivo.
- Cálculo de porcentaje en tiempo real.
- Historial Automático: Los resultados del día se archivan automáticamente al cambiar de fecha, permitiendo revisar el desempeño histórico.
- PWA Ready: Configuración inline para funcionar como una aplicación nativa en dispositivos móviles.
- Diseño Responsive: Optimizado para iOS y Android con soporte para "Safe Areas" (notch).

Tecnologías Utilizadas
- HTML5 & CSS3: Uso de variables CSS, Flexbox y animaciones personalizadas.
- JavaScript (Vanilla): Lógica de estado, manipulación del DOM y persistencia de datos.
- LocalStorage: Almacenamiento local para que no pierdas tus datos al cerrar el navegador.
- Google Fonts: Uso de DM Sans para legibilidad y DM Mono para datos numéricos

Instalación y UsoAl ser una aplicación de un solo archivo (Single File App), no requiere servidores ni instalaciones complejas:
- Descarga el archivo habitflow.html.
- Ábrelo en cualquier navegador moderno.
- ¡Empieza a trackear tus hábitos!
Tip: En móviles, selecciona "Añadir a la pantalla de inicio" para usarla como una App nativa sin barras de navegación del navegador.

Interfaz de Usuario
- La aplicación utiliza un esquema de colores Deep Ocean:
- Fondo: #0f0f1a
- Acentos: Violeta y púrpura (#6c63ff, #a78bfa)
- Estados: Verde (Éxito), Amarillo (Progreso), Rojo (Pendiente).
